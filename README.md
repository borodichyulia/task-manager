# Документация к проекту

## Быстрый старт

Клонировать репозиторий
   ```bash
   git clone <https://github.com/borodichyulia/task-manager>
   ```
### Backend 
1. Перейти в папку `back-task-manager`
2. Запустить докер команду, которая поднимет базу данных
    ```bash
    docker-compose up -d
   ```
3. Создать файл `.env` в корне текущей директории и настроить подключение к базе данных
4. Сделать миграцию базы данных `php artisan migrate`
5. Запустить приложение `comoser run dev`

### Frontend
1. Перейти в папку `front-task-manager`
2. Запустить команду `npm install`
3. Запустить приложение `npm start`

## Технические детали
Возможность создания задач незарегистрированным пользователям была реализована на уровне базы данных при помощи избыточности колонок,
тем самым это нарушает 3 нормальную форму, преимущество данного решения - скорость разработки. В продакшен системе я бы такой подход
использовала в крайнем случае. Возможный вариант решения: **Class Table Inheritance Pattern**
